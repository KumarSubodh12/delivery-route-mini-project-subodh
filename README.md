# ADA Capstone Project

## Delivery Route Optimization using Algorithmic Strategies

---

### 📌 Project Overview

This project simulates a real-world delivery system similar to modern e-commerce platforms. It focuses on optimizing delivery routes by combining multiple algorithmic paradigms including Recursion, Greedy Algorithms, Dynamic Programming, Graph Algorithms, and the Travelling Salesman Problem (TSP).

---

### 🎯 Objectives

* Model delivery routing as an optimization problem
* Apply multiple algorithmic strategies together
* Minimize delivery cost and maximize efficiency
* Analyze performance and scalability of algorithms

---

### ⚙️ Algorithms Implemented

#### 🔹 Recurrence (Route Cost Calculation)

* Recursive approach to calculate total route cost
* Explores all possible delivery paths

---

#### 🔹 Greedy Algorithm

* Selects parcels based on value-to-weight ratio
* Fast and efficient decision-making

---

#### 🔹 Dynamic Programming

* Solves optimization problems like knapsack
* Helps in delivery planning and time constraints

---

#### 🔹 Graph Algorithms

* **Dijkstra Algorithm** → Shortest path between locations
* **Minimum Spanning Tree (MST)** → Minimum cost network design

---

#### 🔹 Travelling Salesman Problem (TSP)

* Finds optimal delivery route
* Demonstrates NP-hard complexity

---

### 📊 Performance Analysis

* Execution time measured using Python `time` module
* Observed exponential growth in TSP
* Compared efficiency of different strategies

---

### ▶️ How to Run

### 📌 Key Observations

* Greedy algorithms are fast but may not give optimal results
* Dynamic Programming ensures optimal solutions with higher memory usage
* Graph algorithms efficiently handle routing problems
* TSP becomes infeasible for large inputs due to exponential complexity

---

### 📌 Real-World Applications

* E-commerce delivery systems (Amazon, Flipkart)
* Logistics and supply chain optimization
* Transportation and route planning systems
* Smart navigation and mapping systems

---

### 📌 Conclusion

* Hybrid use of multiple algorithms improves performance
* Trade-offs exist between speed and optimality
* Real-world systems require efficient and scalable solutions
